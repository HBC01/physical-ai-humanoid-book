# Chapter 1: Introduction to NVIDIA Isaac Sim

## Learning Objectives

By the end of this chapter, you will be able to:

- Understand NVIDIA Isaac Sim and its advantages for robotics
- Install Isaac Sim in GPU and CPU-compatible modes
- Create simulation scenes programmatically using USD
- Import robot models (URDF, USD, MJCF)
- Simulate sensors with high fidelity
- Use the Python Scripting API

## Introduction

NVIDIA Isaac Sim is a GPU-accelerated robotics simulator built on Omniverse, offering photorealistic rendering and physics simulation at scale.
