## üéØ Project Overview

**Physical AI & Humanoid Robotics Book** is a comprehensive, open-source technical curriculum designed to bridge the gap between AI theory and embodied robotics. This interactive documentation platform teaches developers, researchers, and students how to build intelligent humanoid robots using industry-standard tools like ROS 2, NVIDIA Isaac, and Vision-Language-Action models.

### Why This Project?

Physical AI is revolutionizing robotics, but learning resources are fragmented across research papers, vendor docs, and blog posts. This project consolidates best practices into a structured, accessible curriculum with:

- **7 progressive modules** from foundations to capstone projects
- **Bilingual support** (English + Urdu) for global accessibility
- **Interactive examples** with code snippets, visualizations, and exercises
- **Production-ready infrastructure** built on Docusaurus for easy contribution

## üöÄ Quick Start

### Prerequisites
- Node.js ‚â•20.0
- npm or yarn

### Installation & Development

```bash
# Clone the repository
git clone https://github.com/HBC01/physical-ai-humanoid-book.git
cd physical-ai-humanoid-book/website

# Install dependencies
npm install

# Start development server
npm start
```

The site will open at `http://localhost:3000` with hot-reloading enabled.

### Build for Production

```bash
cd website
npm run build
npm run serve  # Preview production build locally
```
